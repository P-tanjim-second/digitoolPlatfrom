# DigiTools Platform

## 📋 Project Description

DigiTools Platform is a modern, user-friendly marketplace that provides access to premium digital tools and software solutions. The platform allows users to browse, compare, and purchase various digital tools including resume builders, design kits, content calendars, and more. With an intuitive interface and seamless shopping experience, DigiTools makes it easy for professionals to find and acquire the tools they need to enhance their productivity.

---

## 🛠️ Technologies Used

- **Frontend Framework**: React.js
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Component Library**: DaisyUI
- **State Management**: React Hooks (useState)
- **Package Manager**: npm

---

## ✨ Featured Features

### 1. **Dynamic Product Catalog**
Browse and explore a wide variety of digital tools with detailed product cards displaying images, descriptions, pricing, and special badges (Best Seller, Popular, New). Each product can be easily added to your cart with a single click.

### 2. **Smart Products Cart**
A fully functional products cart system that allows users to:
- Add/remove products from cart
- View cart total price
- Track cart item count in real-time
- Proceed to checkout and clear the cart

### 3. **Interactive Pricing Plans**
Choose from multiple subscription plans (Starter, Professional, Enterprise) with detailed feature comparisons and easy sign-up options to suit different user needs and budgets.
